# 简单介绍一下

&emsp;&emsp;文件是一个单文件PHP应用程序，可以下降到你的服务器上的任何目录，立即创建文件和文件夹库。它支持所有文件类型，并允许您预览图像、视频、音频和代码。

- 单个文件。只需将它放入服务器上的任何目录中即可。
- 实时搜索和排序文件。
- 可实时切换的高级列表和库布局。
- 预览照片的 IPTC、EXIF 和 GPS 地图。
- 使用缓存机制和 CDN 快速加载。
- 高级树菜单导航显示整个目录结构。
- 美丽的代码荧光笔，用于查看和编辑代码格式。
- 可选登录以保护访问。
- 上传，移动，复制和删除文件即将推出！
